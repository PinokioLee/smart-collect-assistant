### 프로젝트 명

Smart Collect Assistant: 취합 요청·작성 가이드·파일 병합 자동화 시스템

### 배경 및 문제 정의

* 현재 상황 (As-Is): [현재 업무/프로세스의 문제점]

현재 업무 중 여러 부서 또는 여러 담당자에게 엑셀 파일 작성을 요청하고, 회신된 파일을 다시 취합하는 부수 업무가 반복적으로 발생하고 있다.

취합 업무는 본 업무가 아님에도 불구하고, 다음과 같은 과정 때문에 많은 시간이 소요된다.

1. 취합 요청 메일 내용을 확인한다.

2. 어떤 항목을 어떻게 작성해야 하는지 분석한다.

3. 작성 대상자들에게 작성 방법을 다시 설명한다.

4. 각 작성자가 엑셀 파일을 작성해서 회신한다.

5. 제출 여부를 수동으로 확인한다.

6. 미제출자에게 다시 메일 또는 연락을 한다.

7. 회신된 여러 엑셀 파일을 하나로 취합한다.

8. 공통 항목이 변경되면 여러 엑셀 파일을 다시 열어 각각 수정한다.

9. 최종 취합 파일의 누락, 오류, 중복 여부를 확인한다.

특히 공통 항목이 변경될 경우, 엑셀 파일이 10개라면 10개 파일을 모두 열어 동일한 내용을 반복 수정해야 한다. 이 과정은 단순 반복 작업이지만, 실수 가능성이 높고 시간이 많이 소요된다.

* 기회 요인: [AI로 해결 가능한 이유]

AI와 자동화 기술을 활용하면 취합 업무의 많은 부분을 자동화할 수 있다.

1. **AI 메일 분석**\
   취합 요청 메일을 AI가 분석하여 요청 목적, 작성 항목, 마감일, 작성 대상자를 추출할 수 있다.

2. **작성 가이드 자동 생성**\
   요청 내용을 바탕으로 작성자가 이해하기 쉬운 작성 방법 안내문을 자동 생성할 수 있다.

3. **자동 메일 발송 및 SMS 알림**\
   작성 대상자에게 메일과 알림을 자동 발송하여 요청 누락을 줄일 수 있다.

4. **제출 현황 자동 추적**\
   회신 메일 또는 업로드된 파일을 기준으로 제출 여부를 자동으로 관리할 수 있다.

5. **미제출자 리마인드 자동화**\
   마감 시간이 가까워졌는데 파일을 제출하지 않은 작성자에게 자동으로 재촉 알림을 보낼 수 있다.

6. **엑셀 공통 항목 일괄 업데이트**\
   여러 엑셀 파일에 공통적으로 들어가는 항목을 한 번의 수정으로 전체 파일에 반영할 수 있다.

7. **최종 취합 및 검증 자동화**\
   회신된 엑셀 파일을 하나의 파일로 병합하고, 누락/오류/중복 여부를 자동 검증할 수 있다.

### 핵심 기능 정의

1. 취합 요청 메일 분석 기능 : AI가 요청 메일을 바탕으로 작성자가 이해하기 쉬운 작성법을 자동 생성한다.

2. 작성 가이드 자동 생성 기능 : 기준 엑셀과 비교해서 수정할 엑셀들 업로드

3. 취합 요청 메일 자동 작성 및 발송 기능 : AI가 작성한 가이드를 바탕으로 작성자에게 보낼 메일 초안을 생성한다.

4. SMS 또는 알림 발송 기능 : 취합 요청 메일 발송과 동시에 작성 대상자에게 SMS 또는 사내 알림을 발송한다.

5. 제출 현황 추적 기능 : 작성자별 제출 여부를 자동으로 확인한다.

6. 미제출자 자동 리마인드 기능 : 마감 시간이 가까워졌거나 마감 시간이 지났는데 파일을 제출하지 않은 작성자에게 자동으로 알림을 보낸다.

7. 엑셀 공통 항목 일괄 업데이트 기능 : 여러 엑셀 파일에 공통적으로 들어가는 항목을 한 번에 수정한다.

8. 엑셀 파일 자동 병합 기능 : 자성자들이 제출한 여러 엑셀 파일을 하나의 최종 취합 파일로 자동 병합한다.

9. 데이터 검증 기능 : 최종 취합 전 엑셀 데이터를 자동으로 검증한다.

10. 최종 결과 보고서(이메일) 생성 기능 : 취합 완료 후 결과 보고서를 자동 생성한다.
