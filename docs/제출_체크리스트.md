## 현재 영상 주의

- 기존 `presentation/08079_이형진_시연영상.mp4`는 12개 시나리오와 분석적 추정 ROI를 사용한 과거 버전이다.
- 새 영상은 14개 동일-capability 비교, 구조 실패 복구 3/3, 사람 ROI 미측정을 반영한 뒤에만 제출한다.

## 핵심 수치 (실측 · `data/benchmark_metrics.json`)
- 오류 검출 **F1 100%** (오탐 0 · 미탐 0, 정답 10건)
- 재현성 **표준편차 0** (10회 반복, 결정론)
- 처리 **500행+/초** (46행 자동 처리 실측, 사람 대비는 실측 전 주장하지 않음)
- 자가교정율 **100%** → 재작업 **60%↓**
- 다중 후보 규칙 탐색이 양식 드리프트 **오탐 1건 회피**
